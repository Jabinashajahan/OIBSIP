# ATM Interface - Java Program

## 📌 Task
This project simulates a basic ATM interface using Java. Users can:
- Check account balance
- Deposit funds
- Withdraw funds
- Exit the session

## 💻 Technologies Used
- Java (JDK 8 or above)
- Console/Terminal

## 🧪 How to Run
### 1. Compile the program:javac ATM.java

### 2. Run the program:java ATM
